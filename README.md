# Мультимедийное учебное пособие: Виртуальная Реальность

Интерактивная платформа для изучения основ VR-разработки на Unity и Unreal Engine.

## 🛠 Технологии
* **Backend**: Python, Flask, SQLAlchemy.
* **Frontend**: HTML5, Tailwind CSS.
* **БД**: SQLite.

## 🚀 Запуск
1. Установите зависимости: `pip install -r requirements.txt`.
2. Запустите сервер: `python app.py`.
3. Перейдите по адресу: `http://127.0.0.1:3000`.

## 📂 Структура
* `app.py` — сервер и БД.
* `templates/` — страницы интерфейса.
* `vr_v16.db` — файл базы данных (создается сам).

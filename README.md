# Мультимедийное учебное пособие: Виртуальная Реальность

Интерактивная платформа для изучения основ VR-разработки на Unity и Unreal Engine[cite: 1, 2].

## 🛠 Технологии
* **Backend**: Python, Flask, SQLAlchemy[cite: 3].
* **Frontend**: HTML5, Tailwind CSS[cite: 1, 2].
* **БД**: SQLite[cite: 3].

## 🚀 Запуск
1. Установите зависимости: `pip install flask flask-sqlalchemy`[cite: 3].
2. Запустите сервер: `python app.py`[cite: 3].
3. Перейдите по адресу: `http://127.0.0.1:3000`[cite: 3].

## 📂 Структура
* `app.py` — сервер и БД[cite: 3].
* `templates/` — страницы интерфейса[cite: 1, 2].
* `vr_v16.db` — файл базы данных (создается сам)[cite: 3].
